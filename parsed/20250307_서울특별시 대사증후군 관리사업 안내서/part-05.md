## Chapter 02 서울시 대사증후군 관리사업  DB시스템 화면별 사용방법

### 0. DB 시스템 로그인 화면

| **구분** | **사용 방법** | **비고** |
|----------|----------------|----------|
| DB 시스템 로그인 화면 | ① 사용자의 아이디와 비밀번호를 입력하여 접속<br><br>※ 주의<br>- 아이디에 설정된 IP와 PC IP가 동일하지 않은 경우<br>  → "로그인실패" 메시지 출력 → 지정된 IP의 PC로 접속<br>- 아이디의 비밀번호가 틀린 경우<br>  → "비밀번호가 n회 틀렸습니다." 메시지 출력 → 비밀번호 확인 |  |


### 0-1. 사용자신청(보건소 관리자 권한 메뉴)

| 구분 | 세부기능 | 비고 |
|---|---|---|
| 사용자신청 | ① 메뉴 위치: 관리자 > 사용자신청<br>※ 보건소 관리자로 지정된 사용자만 메뉴가 보임<br>② “Clear 및 신규” 버튼 클릭(신규 입력 화면 생성)<br>③ 기관명, 소속부서 설정<br>④ 아이디 입력 후, 중복확인 버튼 클릭하여 아이디 중복 여부 확인<br>⑤ 나머지 사항 입력<br> - **직종, 사용자 PC IP 입력에 유의**<br>   · 직종에 따라 상담 권한 발생, 사용자 PC IP는 로그인 PC의 IP가 고정<br>   · 사용자 PC IP는 공백 없이 전부 기입<br>⑥ 신청저장하여 사용자신청 정보 저장<br>⑦ 승인받고자 하는 대상에 따라, 해당 담당자에게 서류 제출(하단참조)<br>  a. 업무관리자가 승인받고자 하는 경우, 서울시로 승인요청 공문발송<br>  b. 업무담당자가 승인받고자 하는 경우, 내부결재 후 보건소 업무관리자가 승인<br>※ 제출서류 목록<br>- DB시스템 사용자 신규(변경) 신청서: 서식 6 참조<br>- 보안서약서: 서식 7 참조 |   |

- 사용자 정보의 PC IP와 로그인한 PC IP가 다를 경우, 보건소 관리자가 사용자 화면에서 변경 필요  
- 하단의 “수정” 또는 “초기화” 버튼 클릭으로 IP 불일치로 인한 로그인 불가 상황 처리 가능
  - a. 담당자만 승인한다고 하는 경우  
  DB 시스템 사용자신청 → 서울시 담당주관에게 공문 발송(제출서류 첨부 필수) → 사용자신청 승인(서울시 담당 주관)

  - b. 업무관리자가 승인한다고 하는 경우  
  DB 시스템 사용자신청 → 보건소 내부결재(제출서류 첨부 필수) → 사용자신청 승인(보건소 업무관리자)

### ※ 주의사항-약물치료군으로 분류되지 않는 약복용자 문제

- 해당 현상 해결 방법(아래 순서를 필히 준수)

1. (기초설문, 검사결과 입력을 한 사람이 진행하는 경우) 기초설문 입력 및 저장 후,
2. 대사증후군 관리 창에서 대상자 정보 새로 불러오기(약복용 여부 반영 과정)
3. 검사결과 입력 후 저장

- 해당 현상 발생 사유

1. 기초설문 내 약복용 여부 체크 후 “저장”한 경우, 대사증후군 관리 > 검사결과 > “약 복용” 항목을 시스템에서 자동 체크함
2. 검사결과의 “약복용” 항목 체크 여부에 따라 약물치료군으로 분류되는데,
3. 사용자 정보를 다시 불러오지 않고 “약복용” 항목이 공란으로 되어있는 “열려있는 대사증후군 관리 창”을 사용하여 검사결과를 저장하면서, “약복용” 값이 반영되지 않은 정보가 저장되어 약물치료군으로 분류되지 않는 상황 발생

## Chapter 03 데이터 수정

### 가. 데이터 수정 목적 및 필요성

- 데이터 오류 발견 및 수정을 통한 DB시스템의 신뢰성 유지
- 지속적인 데이터 품질 관리를 통해, 정확하고 신뢰성 있는 정보를 기반으로 서울시 대사증후군 관리사업 서비스 제공

### 나. 데이터 수정 기준

#### 1) (자치구) 직접 수정 가능 데이터
- 대사증후군 관리 참여자 군판정일 일주일 이내: 전 항목 수정 가능
- 실적과 무관한 데이터
  - 등록자 신상정보(이름, 생년월일 등), 기초설문, 상담관리 등
  - 수정 기한 없음, 권한 제한 없음
- 내소동기, 등록센터: 수정 기한 없음, 자치구 보건소 관리자(권한)만 수정 가능
- 검사결과: 검사일로부터 14일 이내 수정 가능, 권한 제한 없음
- 케어플랜: 군판정일로부터 60일 이내 수정 가능, 권한 제한 없음

| **데이터 구분** | **수정 기한** | **수정 권한자** |
|------------------|----------------------|------------------|
| 실적과 무관한 데이터<br>- 등록자 신상정보<br>- 기초설문<br>- 상담관리 등 | 기한 없음 | 제한 없음 |
| 대상자 등록정보 중<br>- 내소동기<br>- 등록센터 | 기한 없음 | 자치구 보건소 관리자(권한) |
| 검사결과 | 검사일로부터 14일 이내 | 제한 없음 |
| 케어플랜 | 군판정일로부터 60일 이내 | 제한 없음 |

※ 기초설문, 검사결과 등 수정 시, 수정한 데이터만 변경될 뿐, 군판정 등 타 데이터는 수정 되지 않음에 유의(군판정, 군분류 변경은 ‘데이터 수정 요청’ 필요)
